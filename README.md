# 🎬 NUXT MOVIE APP

### < 영화 검색 사이트 >

🗓️ **작업기간** : 2022. 03

👨‍💻 **투입인원** : 1명 (개인 프로젝트)

- [**DEMO**](https://nuxt-movie-app-tests.herokuapp.com/)
- [**Vue3 프로젝트**](https://github.com/plutoin/vue-movie-app)

📒 📒 **주요업무**

- 기존의 Vue 프로젝트를 Nuxt.js 프로젝트로 마이그레이션
- Jest와 Cypress를 활용해 각각 단위 / E2E 테스트 적용
- Heroku 이용해 배포 진행
